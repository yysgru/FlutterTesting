# (｡ᵕ ◞ _◟) Commands for Flutter projects in VS Code

  1 - Create: flutter create project_name
  
  2 - Execute: flutter run
  
  3 - Check for available devices: flutter devices
  
  4 - Update dependencies: flutter pub get
  
  5 - Check Flutter version: flutter --version
  
  6 - Check installation state and dependencies: flutter doctor
